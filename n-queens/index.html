Algorithm Overview:

Problem: Place N queens on an NxN chessboard so that no two queens attack each other.

Key Insight: Queens attack along rows, columns, and diagonals. We place one queen per row.

Base Case: When row == N, all queens are placed successfully.

Algorithm Code

int solveNQueen(int row)
{
    // Base case: all queens placed
    if (row == N)
        return 1;

    for (int col = 0; col < N; col++)
    {
        if (isSafe(row, col))
        {
            board[row][col] = 1;

            if (solveNQueen(row + 1))
                return 1;

            // Backtrack
            board[row][col] = 0;
        }
    }
    return 0;
}
